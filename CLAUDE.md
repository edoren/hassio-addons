# Claude Instructions

## Commits

- Use conventional commits (`fix:`, `feat:`, `docs:`, `chore:`, etc.)
- Do not add any Claude signature or link to the chat session in commit messages

## Addon Versioning

- Do not bump the version in `config.yaml` — versioning is handled manually by the maintainer
- When updating `CHANGELOG.md`, follow the [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) format already in use
